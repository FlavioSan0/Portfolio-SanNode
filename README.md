# Portfólio San_Node

Portfólio profissional desenvolvido para apresentar projetos, cases, tecnologias e soluções digitais criadas pela San_Node.

## Visão geral

Este projeto reúne sites, landing pages, sistemas web, automações e produtos digitais desenvolvidos com foco em design moderno, responsividade, performance e experiência do usuário.

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel

## Funcionalidades

- Página inicial com apresentação profissional
- Seção de projetos e cases
- Cards com tecnologias utilizadas
- Links para projetos publicados
- Layout responsivo
- Identidade visual tech clean da San_Node

## Projetos apresentados

- MesaFlow
- AgendaNode
- Sistema financeiro
- Landing pages institucionais
- Sites comerciais
